4、功能说明
TM1628驱动库：针对芯片TM1628操作，主要是面对寄存器操作
TM1628驱动库的应用库：基于上述TM1628电路图和驱动库的键盘显示应用，主要是应用层面操作，
           面对的是7个共阴8段和2个共阳7段LED显示器（后两个还可以当作14个指示灯，逐位操作）
		   以及最多20个按键。
